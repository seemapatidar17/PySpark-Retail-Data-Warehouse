# PySpark Retail Data Warehouse Architecture

## Data Flow

Raw CSV Files
↓
PySpark ETL Pipeline
↓
Data Cleaning & Transformation
↓
Fact and Dimension Tables
↓
Delta Lake Storage
↓
SQL Analytics Layer
↓
Databricks Dashboard

### Data Source

* Olist Brazilian E-Commerce Dataset
* CSV Files

### ETL Layer

* PySpark
* Schema Inference
* Data Validation
* Data Transformation

### Storage Layer

* Delta Lake Tables

### Analytics Layer

* SQL Queries
* KPI Calculations
* Revenue Analysis

### Visualization Layer

* Databricks Dashboard

* Revenue Charts
